# Project Evaluation / Evaluation projet

[EN](#english) | [FR](#francais)

## English

### How To Judge The Project

Start with one workflow and ask whether the result remains understandable after the run. A strong result shows the request, constraints, tool path, artifact, checks, uncertainty, and human decision.

That is the product test for Codex Model Orchestrator.

### What To Review First

For the main project, review [Codex Model Orchestrator](projects/codex-model-orchestrator.md) and [QA validation](qa-validation.md). For concrete workflow surfaces, review [CodexToUnity](projects/codextounity.md), [Mob'ia / ccomf-unity](projects/mobia-ccomf-unity.md), and [LocalAssetFactory](projects/local-asset-factory.md).

### Strong Signals

- The run explains itself without a long voice-over.
- The artifact is named and reviewable.
- The checks match the surface: tool state, Unity import, ComfyUI job, or local asset candidate.
- The decision is visible.
- The next action is small and concrete.

### Useful Support Areas

MCP/App SDK review, run-summary UX, proof-card design, Unity handoff review, ComfyUI job lifecycle, local asset QA, documentation, funding, mission work, and product/tooling roles.

## Francais

Commencer par un workflow et demander si le resultat reste comprehensible apres le run. Un resultat fort montre demande, contraintes, chemin outil, artefact, controles, incertitude et decision humaine.

C'est le test produit pour Codex Model Orchestrator.

### Quoi Reviewer En Premier

Pour le projet principal, lire [Codex Model Orchestrator](projects/codex-model-orchestrator.md) et [QA validation](qa-validation.md). Pour les surfaces de workflow concretes, lire [CodexToUnity](projects/codextounity.md), [Mob'ia / ccomf-unity](projects/mobia-ccomf-unity.md) et [LocalAssetFactory](projects/local-asset-factory.md).

### Signaux Forts

- Le run s'explique sans longue presentation orale.
- L'artefact est nomme et reviewable.
- Les controles correspondent a la surface: etat outil, import Unity, job ComfyUI ou candidat asset local.
- La decision est visible.
- La prochaine action est petite et concrete.

### Zones De Support Utiles
