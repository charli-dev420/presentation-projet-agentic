# Iconography / Iconographie

[EN](#english) | [FR](#francais)

## English

Use icons to identify workflow surfaces quickly.

- Unicorn mark: Unicorn Who Dev identity.
- Flow arrows: orchestration and handoff.
- Check mark: validated check or written result.
- Folder: artifact, manifest, or project package.
- Cube: Unity asset candidate.
- Phone/window: Mob'ia client surfaces.
- Plug: MCP tool contract or connected capability.
- Shield: security/privacy review.

## Francais

Utiliser les icones pour identifier rapidement les surfaces de workflow.

- Marque licorne: identite Unicorn Who Dev.
- Fleches de flux: orchestration et handoff.
- Check mark: controle valide ou resultat ecrit.
- Dossier: artefact, manifest ou package projet.
