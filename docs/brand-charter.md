# DAWWW Core Brand Charter / Charte DAWWW Core

[EN](#english) | [FR](#francais)

## English

### Available Assets

| Asset | Use |
| --- | --- |
| `assets/brand/daw-core-logo-main.png` | Main logo for repository identity and presentation material. |
| `assets/brand/daw-core-icon-512.png` | Compact icon for README and small surfaces. |
| `assets/brand/daw-core-banner.png` | Wider banner when a visual header is useful. |
| `assets/brand/daw-core-watermark-opacity20.png` | Quiet watermark for public-safe diagrams or proof material. |

### Agentic Use

- Use the brand to identify the DAWWW Core presentation hub.
- Keep the tone technical, compact, and evidence-led.
- Do not use the brand to imply a proof level that has not been reached.
- Connect claims to evidence levels, QA gates, or explicit public scope.
