# Brand Charter / Charte de marque

[EN](#english) | [FR](#francais)

## English

### Available Assets

- `assets/brand/daw-core-logo-main.png` - main Unicorn Who Dev identity mark used across showcase material.
- `assets/brand/daw-core-icon-512.png` - compact icon for avatars, favicons, and badges.
- `assets/brand/daw-core-banner.png` - wide cover banner.
- `assets/brand/daw-core-watermark-opacity20.png` - light watermark for diagrams and proof material.

### Usage Rules

Use the main logo as an identity signal, not repeated decoration. Keep diagrams readable. Use banners to separate product lines: Codex Model Orchestrator first, then Unity/ComfyUI and asset workflows.

### Tone

Concrete, product-oriented, and evidence-aware. Prefer language that explains what the workflow does, how it is checked, and what help would be useful next.
