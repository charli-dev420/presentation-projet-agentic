# Public Scope / Perimetre public

This repository is a public-safe presentation surface.

## Included

- High-level product positioning.
- Public-safe architecture principles.
- Non-sensitive diagrams.
- Partnership and funding needs.
- Security and governance principles.

## Excluded

- Source code.
- Private MCP servers, plugins, prompts, traces, datasets, or logs.
- API keys, tokens, local endpoints, private configuration, or user data.
- Security scan artifacts, release gates, internal run summaries, or operational evidence.
- Any tool definition that would expose destructive capabilities or private workflows.

## Review rule

Only add material that can be shared publicly without exposing credentials, private tools, privileged workflows, or customer/user data.
