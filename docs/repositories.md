# Repositories Covered / Repos couverts

[EN](#english) | [FR](#francais)

## English

This showcase represents several real project families through presentation material.

**Codex Model Orchestrator** is the main project family: orchestration, routing, validation, MCP/App SDK concepts, proof-card language, and run summaries.

**CodexToUnity** is the public bridge between Codex, Unity, and ComfyUI, focused on asset handoff and Unity review. Public route: [Unicorn Who Dev / codextounity](https://github.com/charli-dev420/codextounity).

**Mob'ia / ccomf-unity** is the product layer around profiles, jobs, artifacts, clients, and ComfyUI review state.

**LocalAssetFactory** is the local asset loop for generation, manifest, normalization, Unity import, socket mapping, scene review, and written evidence.

The repo publishes product explanation, diagrams, resources, QA vocabulary, and presentation assets. It does not need to reproduce implementation work to be useful.
