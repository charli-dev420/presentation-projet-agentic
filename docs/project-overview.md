<h2 id="english">🇬🇧 English</h2>

This page gives a clean overview of the agentic project family presented in this repository.

The repository is not limited to one product. It groups projects where a language model has a central role in the workflow: reading, planning, routing, generating, inspecting, editing, evaluating or preparing a human decision.

Some projects also appear in Unity, 3D, music or creative-tool showcases. Here, the focus stays on the LLM layer.

---

## Direction

The shared direction is simple: an AI-assisted result should not become a black box.

A useful workflow should make it possible to understand:

- what was requested;
- what the LLM was expected to do;
- which tools or surfaces were involved;
- what artifact, state or note was produced;
- what still needs review;
- what decision comes next.

## Public boundary

This public repository presents the map, the positioning and the review logic.

It does not expose private source code, credentials, private prompts, unreleased binaries, sensitive automation details or internal proof archives.
