This page tracks the public visual material for the agentic showcase.

The visuals should make the LLM-centered map easier to understand without exposing private workflows or raw implementation material.

| --- | --- | --- |
| Agentic workflows banner | Main showcase identity. | High |

| LLM role diagram | Explains read, plan, route, generate, inspect, edit, evaluate and decide. | High |

## Missing public visuals

Useful next visuals:

- one global map of the project family;
- one diagram of LLM roles;
- one screenshot or mockup per main project line;
- one before / after style visual for Codex Image Génial;
- one preset evaluation flow for uwdevst_hub;

## Visual boundary

Public visuals should be clean and explanatory.

They should avoid private screenshots, sensitive paths, credentials, unreleased customer data, raw prompts, private datasets or internal proof archives.
