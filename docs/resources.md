## Francais

### Glossaire

| Terme | Definition |
| --- | --- |
| Agentic | Systeme ou l'agent planifie, utilise des outils et produit des preuves. |
| MCP | Protocole de connexion entre agent et outils. |
| Apps SDK | Surface pour rendre des interfaces lisibles dans une experience agentique. |
| Proof kit | Ensemble de documents et artefacts rediges pour qualifier les claims. |
| BYOK | Bring your own key, utile pour benchmarks communautaires. |
| ComfyUI | Outil node-based pour workflows de generation IA. |
| Trellis / GLB | Familles de generation et format d'asset 3D, selon le pipeline. |
