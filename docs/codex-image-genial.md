<h2 id="english">🇬🇧 English</h2>

Codex Image Génial is the Codex-assisted image editing direction.

It can also appear in an image or creative-tools showcase. In this repository, the focus is the LLM role inside the editing loop: describe, modify, script, compare and review changes from the discussion.

---

## LLM role

The LLM can help:

- understand the requested edit;
- turn a visual request into structured operations;
- prepare code-assisted changes;
- compare before / after intent;
- explain what changed;
- support iterative review.

---

## Useful workflow

```text
image request → edit plan → code-assisted operation → result → comparison → revision note
```

The goal is to make image editing more controllable from the conversation without hiding the review step.
