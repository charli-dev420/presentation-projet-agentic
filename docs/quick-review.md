<h2 id="english">🇬🇧 English</h2>

Use this page for a short first review of the repository.

---

## Five-minute path

1. Read the README to understand the LLM-centered positioning.
2. Read [`agentic-projects.md`](agentic-projects.md) to see the project map.
3. Read [`llm-roles.md`](llm-roles.md) to understand what the LLM actually does.
4. Pick one project page, not all of them.
5. Check whether the role, workflow and public boundary are clear.

---

## Review questions

A useful first review answers:

- Is the agentic angle clear?
- Is the LLM role understandable?
- Does any project sound overpromised?
- Is the documentation path simple enough?
- Which project should be expanded first?
