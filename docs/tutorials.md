# Tutorials / Tutoriels

[FR](#francais) | [EN](#english)

## Francais

### Tutoriel 1 - Comprendre MCP sans code

MCP sert de contrat entre l'agent et des outils. Un bon contrat indique ce que l'outil peut lire, ce qu'il peut modifier, quelles confirmations sont requises et quelles preuves seront produites.

### Tutoriel 2 - Lire une preuve agentic

Une preuve utile indique: objectif, contexte, outils appeles, resultats, limites, statut, et ce qui a ete redige. Une trace brute ou un prompt complet n'est pas une preuve publique acceptable.

### Tutoriel 3 - Evaluer un job asset

1. Identifier l'intention.
2. Verifier le perimetre autorise.
3. Suivre l'etat du job.
4. Lire les sorties partageables.
5. Valider l'import ou documenter l'echec.

### Tutoriel 4 - Preparer une collaboration

Avant une collaboration, definir: objectifs, donnees autorisees, outils autorises, niveau de preuve attendu, criteres d'acceptation et contenu qui restera prive.

## English

These tutorials explain MCP as a tool contract, public-safe agentic evidence, asset-job evaluation, and collaboration preparation without exposing private prompts, traces, configs, or workflows.
